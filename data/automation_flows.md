# Automation Flows -- Pinterest Income Assistant

## Pinterest Income Automation Flow

1.  User inputs (niche, product, keywords, goals)
2.  AI retrieves data from /data (RAG)
3.  AI generates:
    -   Pin titles
    -   Descriptions
    -   Keywords
    -   Affiliate scripts
4.  AI optimizes content
5.  AI outputs formatted results

------------------------------------------------------------------------

## Affiliate Marketing Flow

Input: product name → target audience → niche → price

Output: - Affiliate script - Hooks - Long description - Keywords -
Caption - CTA

------------------------------------------------------------------------

## Pinterest Posting Automation Flow

1.  User selects niche
2.  AI generates 14-day content plan
3.  AI generates pin content
4.  AI recommends posting times
5.  AI exports plan (text/csv)
6.  User uploads to Pinterest
